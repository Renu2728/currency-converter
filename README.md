# Currency Converter

A React TypeScript application for converting currencies with react-time exchange rates.

# Features

1.Convert between 150+ currrencies
2.Searchable currency dropdown with flags
3.Input validation with error message
4.Swap currencies with one click
5.10-minutes countdown timer
6.Fully responsive design

## Quick Start

# Install dependencies

npm install

# Start development server

npm start
App run on http://localhost:3000/

# Run tests

npm test
